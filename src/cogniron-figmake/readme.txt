== Description ==

Cogniron Figma Make Theme is a modern, pixel-perfect block theme (Full Site Editing) converted from a professional Figma design. 

= Features =

* Full Site Editing (FSE) - Block Theme
* Pixel-perfect design matching the Figma prototype
* AI-powered QA services showcase
* Comprehensive blog functionality
* Fully responsive across all devices
* Glassmorphism header with backdrop blur
* Subtle grid pattern in hero section
* Built-in animations and hover effects
* Custom color palette (#E2F2FF, #0054A6, #00C2C7)
* Complete spacing system (96px, 64px, 48px, 32px, 24px, 16px)
* Dark and light theme support
* Accessibility ready
* Translation ready
* SEO optimized

= Theme Structure =

* Full Site Editing templates (front-page, index, single, page, archive, etc.)
* Reusable block patterns (hero, services, features, testimonials, CTA, blog grid)
* Custom template parts (header, footer)
* Comprehensive CSS with animations
* JavaScript for interactivity and scroll effects
* Placeholder images for easy replacement

= Installation =

1. Upload the theme ZIP file via Appearance → Themes → Add New → Upload Theme
2. Activate the theme
3. Go to Appearance → Editor to customize templates and patterns
4. Replace placeholder images via the media library
5. Edit text content directly in the Site Editor

= Customization =

All text, images, and layouts are editable through WordPress:
* Use the Site Editor (Appearance → Editor) to modify templates
* Replace images by uploading new ones in the block editor
* Change colors via theme.json settings
* Customize navigation menus in Appearance → Menus

= Blog Usage =

Simply create new blog posts from the WordPress dashboard. They will automatically appear in:
* Blog archive page (/blog)
* Homepage latest insights section
* Category and tag archives
