# Cogniron Pro - Image Assets

## Placeholder Images

This theme uses placeholder images that should be replaced with your actual images. Below are the specifications for each placeholder image:

### 1. hero-placeholder.jpg
- **Dimensions:** 1920 x 1080 pixels (16:9 aspect ratio)
- **Usage:** Hero section background or featured image
- **Recommended Content:** Modern office workspace, technology devices, or AI/QA testing visualization
- **File Size:** Keep under 500KB (optimized)

### 2. logo-placeholder.png
- **Dimensions:** 300 x 100 pixels (recommended)
- **Usage:** Site logo in header
- **Recommended Content:** Your company logo with transparent background
- **File Format:** PNG with transparency or SVG for scalability
- **File Size:** Keep under 50KB

### 3. placeholder.jpg
- **Dimensions:** 1200 x 630 pixels (recommended for general use)
- **Usage:** General placeholder for blog posts, services, case studies, etc.
- **Recommended Content:** Relevant images for your content (team photos, product screenshots, etc.)
- **File Size:** Keep under 300KB (optimized)

## How to Replace Placeholder Images

### Option 1: Via WordPress Admin Dashboard
1. Go to **Appearance → Customize**
2. Navigate to the relevant section (Hero Section, Services, etc.)
3. Click on the image upload button
4. Upload your image from your computer
5. Click **Publish** to save changes

### Option 2: Direct File Replacement
1. Prepare your images according to the specifications above
2. Name them exactly as shown: `hero-placeholder.jpg`, `logo-placeholder.png`, `placeholder.jpg`
3. Upload them to `/wp-content/themes/cogniron-pro/assets/images/`
4. Replace the existing files

### Option 3: Via WordPress Media Library
1. Go to **Media → Add New** in WordPress admin
2. Upload your images
3. Use the uploaded images in your pages/posts via Gutenberg blocks
4. For hero/services sections, use Customizer to select from Media Library

## Image Optimization Tips

- Use tools like TinyPNG, ImageOptim, or ShortPixel to compress images
- For hero images, consider using WebP format for better compression
- Always set appropriate `alt` text for accessibility
- Use lazy loading for images below the fold (theme handles this automatically)

## Featured Images for Blog Posts

When creating blog posts, always set a Featured Image:
1. Edit your post in WordPress
2. Look for "Featured Image" in the right sidebar
3. Click "Set featured image"
4. Upload or select an image (recommended: 1200 x 675 pixels)

## Custom Logo

To upload your custom logo:
1. Go to **Appearance → Customize → Site Identity**
2. Click "Select logo"
3. Upload your logo file (PNG with transparency recommended)
4. Adjust the size as needed
5. Click **Publish**
